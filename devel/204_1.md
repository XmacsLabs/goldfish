# isocline-based REPL

## 如何测试？

启用 repl option 后编译
```sh
xmake f -v -y --repl=true
xmake b goldfish
```

使用 repl 测试
```sh
./bin/goldfish -i
```

## 2025/07/29

### What

新增 isocline xmake 本地仓库，默认启用 isocline-based REPL。

在 `src/goldfish.hpp` 中新增 isocline-based REPL 实现。

### Why

isocline 较 readline 更适合在 windows 上编译，跨平台能力更强。且自带 highlight，completion 等功能。
