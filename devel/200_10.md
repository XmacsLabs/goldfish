# [200_10] 初始化 nix flake 支持

## 如何测试？

使用 nix 构建 goldfish
```sh
nix build .#goldfish .#goldfish-musl
```

进入 nix devshell
```sh
nix develop
```

如果有 direnv，可以使用 direnv 自动进入 devshell
```sh
direnv allow
```

全仓库格式化
```sh
nix fmt
```
