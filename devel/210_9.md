## 2026/1/27 实现 SRFI-267，并基于此实现 `(liii raw-string)`

### What

实现 SRFI-267 与 `(liii raw-string)`。

SRFI-267 提供了一种 raw string 的字面量写法：
```
 ⟨raw string (X)⟩ ⩴ #" X " ⟨raw string internal (X)⟩ " X "
```

摘抄示例如下：
```
#"""" ; → ""
#""a"" ; → "a"
#""\"" ; → "\\"
#"-"""-" ; → "\""
#"-" " "-" ; → " \" "
#"-"#""a"""-" ; → "#\"\"a\"\""
#"-"ends with \""-" ; → "ends with \\\"
#""multiline
string"" ; → "multiline\nstring"
#""
    no whitespace stripping"" ; → "\n    no whitespace stripping"
#""\(?(\d{3})\D{0,3}(\d{3})\D{0,3}(\d{4})""
  ; → "\\(?(\\d{3})\\D{0,3}(\\d{3})\\D{0,3}(\\d{4})"
  ; Example from SRFI 264
```

`(liii raw-string)` 提供了一个宏 `deindent` (alias `&-`)，用于 C# 风格的缩进对齐。

注：SRFI 还处于 Draft 状态，有可能更改。

测试与代码实现中有 raw string 字面量的需求。
