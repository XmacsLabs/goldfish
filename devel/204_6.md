# [204_6] 修复补全项的错误分割

## 如何测试？

进入 REPL 尝试补全

修复前：
```sh
./bin/goldfish -i
...
gf> call/<tab>
 1 @      4 ???    7 *s7*
 2 gc     5 eq?    8 any?
 3 pi     6 map    9 eqv?
(press page-down (or ctrl-j) to see all further completions)
gf>
```

这里将 `/` 视作 token 的分割了，因此没有将 `call/` 当作 token 的前缀来匹配。

修复后：
```sh
./bin/goldfish -i
...
gf> call/<tab>
;; 补全为
gf> call/cc
gf>
```

正确补全
